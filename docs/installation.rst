============
Installation
============


Stable release
--------------

To install simclr_pytorch_lightning, run this command in your terminal:

.. code-block:: console

    $ pip install simclr_pytorch_lightning

This is the preferred method to install simclr_pytorch_lightning, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for simclr_pytorch_lightning can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/aced125/simclr_pytorch_lightning

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/aced125/simclr_pytorch_lightning/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/aced125/simclr_pytorch_lightning
.. _tarball: https://github.com/aced125/simclr_pytorch_lightning/tarball/master
